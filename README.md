# Farouk's Blog - Site Vitrine

A beautiful, responsive showcase website for a personal blog built with vanilla HTML, CSS, and JavaScript.

## 🌟 Features

- **Login Page**: Elegant login interface with teal gradient background
- **Blog Page**: Clean blog layout with card-based design
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Vanilla Tech Stack**: No frameworks - pure HTML5, CSS3, and JavaScript
- **Modern Design**: Based on Figma design specifications
- **Accessibility**: Semantic HTML, ARIA labels, and keyboard navigation support
- **Performance Optimized**: Fast loading times with smooth animations

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A local web server (optional, but recommended for testing)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Hazem-Barkallah/Farouk-blog.git
cd Farouk-blog
```

2. Open the website:
   - **Option 1**: Simply open `index.html` in your browser
   - **Option 2**: Use a local server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```
   - Then navigate to `http://localhost:8000`

## 💻 Usage

### Login Page
1. Open `index.html` in your browser
2. Enter any password (minimum 3 characters for demo)
3. Click "LOGIN" to access the blog

### Blog Page
- Browse through the blog post cards
- Click on any card for interaction effects
- Fully responsive across all devices

## 🎨 Design Features

### Color Scheme
- **Login Background**: Teal gradient (#4db8a8 to #2a9d8f)
- **Blog Cards**: Warm beige (#E8C9A1)
- **Typography**: Dark gray (#2b2b2b) for text

### Typography
- **Headings**: Cormorant Garamond (elegant serif)
- **Body**: Poppins (clean sans-serif)

### Responsive Breakpoints
- **Mobile**: < 480px
- **Tablet**: 481px - 768px
- **Desktop**: > 769px
- **Large Screens**: > 1400px
